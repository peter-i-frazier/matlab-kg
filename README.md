# matlab-kg

Matlab KG library from two early papers on the knowledge gradient method:
- Frazier, Powell & Dayanik. "A knowledge-gradient policy for sequential information collection", SIAM Journal on Control and Optimization, 2008
- Frazier, Powell & Dayanik. "The knowledge gradient policy for correlated normal beliefs", INFORMS Journal on Computing, 2009

The code is written in Matlab.
